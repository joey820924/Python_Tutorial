# 數值資料

## 資料型態

Python3中的數值型態有三種：  
1. 整數\(int\)  
2. 浮點數\(float\)  
3. 複數\(complex\)  
  
在Python3中，整數並無分為整數與長整數，長度並無限制\(但在硬體限制內\)。整數的進位制整理如下

```
$ give me super-powers
```
